# TMS-Angu

## Tech Stack
- .NET 8 Web API
- SQL Server (Stored Procedures)
- Angular / React
- Dapper / EF Core

## Setup Instructions

### Backend
1. Update connection string in appsettings.json
2. Run: dotnet run

### Database
1. Run table_schema.sql
2. Run stored_procedures.sql

### Frontend
1. npm install
2. npm start

## Features
- CRUD operations
- Stored procedure implementation
- Clean architecture
